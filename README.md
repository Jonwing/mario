# mario

A command line tool for managing multiple SSH tunnels interactively.

## Features

 + handle multiple tunnels concurrently
 + auto reconnect
 + auto complete
 + save tunnels to disk
 
## Installation
 
```bash
go get -u github.com/Jowing/mario
```
